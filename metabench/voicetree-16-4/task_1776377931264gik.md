## The fix (already diagnosed — just apply)

Cell 4 currently has:
```python
result = _t.evaluate(kbench.llm, evaluation_data=df)
```

Replace that ONE line with:
```python
result = _t.evaluate(evaluation_data=df, llm=[kbench.llm])
```

Why: `Task.evaluate(grid, evaluation_data, ...)` — the first positional is a param-grid dict, not llm. Passing `kbench.llm` (an OpenAI client) makes kbench try `openai_client |= kwargs` → TypeError. Passing llm as a 1-element list via kwargs avoids the `|=` path (first branch `grid = kwargs` executes instead).
